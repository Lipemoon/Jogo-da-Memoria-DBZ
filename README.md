NOME DO PROJETO: JOGO DA MEMORIA DRAGON BALL

INTEGRANTES: Leonardo da Silva Martinelle e Luis Felipe Nunes 

DESCRIÇÃO: Um jogo de memória inspirado em personagens do Dragon Ball, onde o jogador tem 
que achar os pares iguais das imagens. 

API UTILIZADA E JUSTIFICATIVA: 
API de Dragon Ball -> https://web.dragonball-api.com/
A justificativa é porque o jogo é sobre Dragon Ball

INTRUÇÕES DE USO: Abrir o arquivo HTML, e jogar o jogo

Conceitos de POO implementados e onde encontrá-los no código: Na criação de objetos em javascript, em funções reutilizadas em lógicas no código.
